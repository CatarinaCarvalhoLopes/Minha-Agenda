# Minha Agenda 🗓️

Aplicativo web para organizar compromissos de forma simples, rápida e acessível.

## ✨ Funcionalidades

- Login com Google
- Criação, visualização e gerenciamento de compromissos
- Interface clara e responsiva
- Tudo salvo em nuvem com Firebase
